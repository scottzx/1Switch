# Dieter Rams 10 Principles

1. **Good design is innovative** - It does not copy existing solutions
2. **Good design makes a product useful** - Function first, aesthetics second
3. **Good design is aesthetic** - Beauty serves usability, not decorative
4. **Good design makes a product understandable** - Clear hierarchy, obvious function
5. **Good design is unobtrusive** - Products fulfill a purpose, like tools
6. **Good design is honest** - Does not attempt to make product more innovative, powerful or valuable than it actually is
7. **Good design is long-lasting** - Avoids being fashionable, never dates
8. **Good design is thorough down to the last detail** - Nothing is left to chance
9. **Good design is environmentally friendly** - Conserves resources
10. **Good design is as little design as possible** - Less, but better

## Application to iClaw Design System

- **Neutral palette**: No flashy colors, warm/cool neutrals only
- **Brass accent**: Single warm accent color (reference: Braun TP1)
- **Grid system**: Strict 16px grid, 12-column layout, 1200px max-width
- **Minimal shadows**: Only functional shadows, no decorative glows
- **Typography hierarchy**: Clear size steps, uppercase labels with tracking
- **Transitions**: Subtle 150-200ms, never bouncy or flashy
